Detailed Description

Klass som representerar en pilbågs fiende i spelet.

Ärver från Enemy och använder funktioner från Enemy. Innehåller funktioner för att hantera fiendens beteende och interaktion med spelaren.

Konstruktor för ArrowEnemy.

Initierar pilbågs fienden med två olika hitboxes. En för när fienden kolliderar, den andra för när fienden kan upptäcka spelaren.

Parameters
cordKordinaterna där objektet ska skapas.
objectAnvänds för att hämta rätt textur för rätt objekt.
